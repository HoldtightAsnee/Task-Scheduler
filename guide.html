Task Scheduler

Creating a schedule

  • Select the 'Start Day'. This is the first day of the schedule.
  • Select the 'End Day'. This is the last day of the schedule. The days between the start day
    and the end day will be populated by the scheduler. To create a schedule for a single day,
    set the end day to be the same as the start day.
  • Set the 'Start Time'. This is the time you would like your schedule to begin.
  • Set the 'End Time'. This is the time you would like your schedule to end.
  • Set the 'Duration of Each Task'. This should be a time between the 'Start Time' and 'End Time'.
    The scheduler will populate a suitable number of tasks with this duration.

Adding Days

  • Add Day Before: Adds the day before the current day if it is not already in the schedule.
  • Add Day After: Adds the day after the current day if it is not already in the schedule.
  • A day can only be added if it is not already in the schedule, and there are no more than 7 days in the schedule.

Removing Days

  • Remove Day: Removes the current day from the schedule. All tasks on this day will be deleted.

Adding Tasks

  • Adding tasks creates a new task with the duration set to the value of 'Duration of Each Task' which was set when creating the schedule.

Editing Tasks

  • Changing the task description: The description can be changed without having to edit the task.
  • Changing the 'Start Time': This time cannot exceed nor equal the ending time of the task.
    The scheduler adjusts the 'Duration' of the task after the 'Start Time' has been changed.
  • Changing the 'End Time': This time cannot be before nor equal to the starting time of the task.
    The scheduler adjusts the 'Duration' of the task after the 'End Time' has been changed.
  • Changing the 'Duration': This time cannot be equal to '00:00'.
    The scheduler adjusts the 'End Time' after the 'Duration' has been changed.
  • To confirm the edit, press the 'Save' button.

Removing Tasks

  • Removing a task deletes that it from the schedule but does not break the your scheule's flow.
    The times of the tasks after are adjusted.

Saving Schedule

  • To save the schedule, save it as a pdf on your device.

Copying Days

  • Select the day you would like to copy by pressing the 'Copy' button on the desired day.
  • Press 'Paste' on the day which you would like to be overwritten.
  • When done, press 'Cancel' to exit the copy mode.
